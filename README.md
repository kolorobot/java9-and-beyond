# Java 9 and Beyond

Samples of different Java language features from version 9 onward.

# Run

To run all the samples, use the following command:

```bash
./mvnw clean test
```

To run a specific sample, use the following command:

```bash
cd <sample-directory>
../mvnw clean test
```
